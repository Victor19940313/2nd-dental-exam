CHAPTER 1

缺牙的開始

為什麼要做假牙？三條路怎麼選？
明
陳醫師，我左下後面少了兩顆牙，右邊也少了一顆，吃東西都只能用前面咬。到底要怎麼辦？
師
阿明你先別緊張。缺牙不處理的話，旁邊的牙齒會慢慢倒下來、對面的牙齒會往下長（過度萌發），咬合會整個亂掉。我們有三條路可以走。

三大假牙選項

想像你家客廳少了幾片地磚：

生活化比喻
活動假牙（RPD）= 鋪一塊地毯蓋住缺口，每天可以拿起來清洗
固定假牙（FPD）= 重新黏上新地磚，跟旁邊的磚緊緊相連
全口義齒（CD）= 整個地板重鋪，因為一片磚都不剩了
選項英文特點適合誰
活動局部義齒RPD
(Removable Partial Denture)		可自行摘戴，有金屬骨架缺多顆但還有自然牙的人
固定義齒FPD
(Fixed Partial Denture)		黏在支柱牙上，不能自行取下缺 1-2 顆，旁邊牙齒狀況好
全口義齒CD
(Complete Denture)		整副假牙，靠吸附力固位全口無牙的人
你
那植牙呢？
師
植牙（Implant）是第四條路，把鈦金屬螺絲打進骨頭裡當人工牙根，上面接假牙。國考會考一些基本概念，但重點還是在 RPD、FPD、CD 這三大類。我們一個一個來。

拔牙後的骨頭變化

你
牙齒拔掉之後，骨頭會怎麼變？
師
拔牙後骨頭會持續吸收，而且有一個非常固定的規律：唇側/頰側骨板比舌側/顎側吸收更快。因為唇側骨板通常比較薄。結果就是牙嵴會越來越窄，而且位置會往舌側偏移。
骨吸收規律：
上顎：骨吸收以唇/頰側為主 → 牙嵴變窄、往顎側退縮 → 上顎牙弓越來越小
下顎：骨吸收以唇/頰側為主 → 牙嵴變窄、往舌側退縮 → 下顎牙弓也變小
結果：上下顎都縮小，但上顎縮得更快 → 原本正常咬合的人，無牙後容易變成假性 Class III（下顎前突）

RPD 的適應症

  • 缺牙跨距太長（FPD 跨距不宜超過 2 顆 pontic）
  • 游離端缺牙（後方沒有支柱牙可做 FPD）
  • 需要取代缺損的軟硬組織（牙嵴嚴重吸收時）
  • 經濟考量或全身健康不適合手術（無法植牙）

FPD 的適應症與禁忌

  • 缺 1-2 顆，前後都有健康支柱牙（bounded saddle）
  • 支柱牙牙根表面積必須符合 Ante's Law
  • 禁忌：支柱牙牙周嚴重破壞、缺牙跨距太長、青少年牙髓腔大

Ante's Law — 支柱牙選擇的黃金法則

你
怎麼判斷旁邊的牙齒夠不夠力當支柱牙？
師
用 Ante's Law：支柱牙的牙根表面積總和，必須大於或等於被取代牙齒的牙根表面積總和。簡單說，支柱牙的「根基」要比缺的牙「根基」更大，橋才站得穩。
Ante's Law：支柱牙的牙周膜面積總和 ≥ 被取代牙齒的牙周膜面積總和
若支柱牙有牙周病（骨喪失）→ 有效牙根面積減少 → 可能需要增加支柱牙數量
注意：Ante's Law 只是起始判斷標準，實際設計還要考慮牙周狀態、咬合力、Crown-root ratio
📋 考試重點框
  • RPD = 可摘式，有金屬骨架；FPD = 固定式，黏在支柱牙上；CD = 全口無牙用
  • 缺牙不處理的後果：鄰牙傾斜、對咬牙過度萌發、咬合紊亂
  • 全口義齒的咬合目標是「平衡咬合」，天然牙的目標是「MPO」
  • Ante's Law：支柱牙牙根表面積 ≥ 被取代牙齒牙根表面積
  • 拔牙後骨吸收：唇側/頰側吸收較快 → 牙嵴變窄往舌側偏
  • 上顎縮得比下顎快 → 無牙後容易出現假性 Class III 關係
  • RPD 適用：多顆缺牙、游離端缺牙、跨距過長無法做 FPD
  • FPD 禁忌：支柱牙牙周差、跨距太長、青少年牙髓腔大
CHAPTER 2

嘴巴的地圖

Kennedy 分類法 — 幫每張嘴巴取名字
師
阿明，在幫你設計活動假牙之前，我得先幫你的嘴巴「分類」。就像醫院急診有五級檢傷，缺牙也有分類系統，叫做 Kennedy Classification。
你
為什麼要分類？
師
因為不同的缺牙分布，假牙的設計邏輯完全不同。分類就像是地址，告訴技師：這個人的嘴巴長什麼樣。

Kennedy 四大類

記住一個口訣：「雙游、單游、中間夾、前牙缺」

Kennedy 四大分類示意圖（Wikimedia Commons, CC BY-SA 4.0）
由左至右：Class I（雙側游離端）、Class II（單側游離端）、Class III（中間夾）、Class IV（前牙跨中線）
分類缺牙型態口訣
Class I雙側後方都缺牙（雙側游離端）I = 一對游離端
Class II單側後方缺牙（單側游離端）II = 一邊游離
Class III缺牙前後都有牙齒（支持端）III = 中間夾
Class IV前牙橫跨中線缺牙IV = 前牙缺
比喻時間
想像一排書架上的書：
Class I = 書架兩端的書都掉了，只剩中間 → 兩邊都「懸空」
Class II = 只有右邊的書掉了 → 一邊懸空
Class III = 中間抽掉幾本，但兩端還有書夾住 → 「有支撐」
Class IV = 書架正中央那排書全不見了 → 跨越中線

Modification（修飾區）

你
如果一個人又缺後面、又缺前面怎麼辦？
師
好問題！主分類看的是最遠端（最後方）的缺牙區。其他額外的缺牙區叫做「Modification」，只數區數，不數缺幾顆。
例如：右後方缺兩顆（游離端）+ 左邊中間缺一區 + 另一處又缺一區
→ 最遠端 = 右後方游離端 → 主分類 = Class II
→ 額外兩個缺牙區 → Modification 2
→ 結果：Kennedy Class II, Mod. 2

法則白話翻譯

前牙全缺 + 後牙單側缺 → 你可能直覺選 Class IV，但錯！
要看最遠端的缺牙在哪裡 → 如果最遠端是後牙 → 答案是 Class I 或 Class II，不是 Class IV！

Applegate 八大法則 — 臨床應用詳解

你
法則 2 和 3 好容易搞混，第三大臼齒到底要不要算？
師
很簡單：如果第三大臼齒缺了而且不打算補，就不算（法則 2）。但如果第三大臼齒還在，而且要被當作支柱牙，就必須算進分類（法則 3）。
法則法則內容臨床實例
法則 1分類在拔牙後才決定治療計畫中要拔的牙齒，先拔完再分類，否則分類會改變
法則 2第三大臼齒缺且不補 → 不列入缺 #18 #28 但不做假牙 → 不影響分類

法則 4最遠端缺牙區決定主分類右後方游離端 + 左前方缺 → 主分類看右後方 = Class II

分類判斷 Step-by-Step 決策流程

你
如果嘴巴裡有好多個缺牙區，要怎麼一步一步判斷？
師
按照這個流程走，不會錯。
先處理第三大臼齒：缺了不補 → 不算；存在且當支柱牙 → 算進去
找出最遠端（最後方）的缺牙區 → 這就決定了主分類
判斷最遠端缺牙是否為游離端（後方無牙）→ 游離端 = Class I 或 II
如果雙側都是游離端 → Class I；只有單側 → Class II
如果最遠端缺牙前後都有牙 → Class III
如果是前牙跨中線缺牙且無更遠端的缺牙 → Class IV
其餘缺牙區的區數 = Modification 數（Class IV 不計 Mod.）

Class I vs Class II — 支撐方式的本質差異

你
Class I、II 都是游離端，跟 Class III 的設計差很多嗎？
師
差非常多！Class III 前後都有牙齒夾住，是 tooth-supported（牙齒支撐型）。但 Class I、II 後面沒有牙可以靠，義齒坐在軟組織上，是 tooth-tissue supported（牙齒+黏膜支撐型），設計難度高很多。
特性Class I / II（游離端）Class III（支持端）
支撐型態Tooth-tissue supportedTooth-supported
義齒動態受力時會旋轉下沉（義齒沉入黏膜）幾乎不旋轉（前後都有支撐）
卡環設計RPI 系統（I-bar + 近心 Rest）圓臂卡環（Aker clasp + 遠心 Rest）
間接固位體必須有通常不需要
印模技術可能需要 Altered Cast一般印模即可
⚠ 常見考法
「Kennedy Class III 需要間接固位體」→ 通常不需要！因為 Class III 前後都有牙齒支撐，不會翹起。
「Kennedy Class I 最需要間接固位體」→ 正確！雙側游離端最容易翹起。
📋 考試重點框
  • Kennedy 分類看最遠端缺牙區決定主分類（Applegate 法則 4）
  • Class IV 沒有 Modification（法則 6，鐵則）
  • Modification 數「區」不數「顆」（法則 7）
  • 不存在 Class V（法則 8）
  • Class I/II = tooth-tissue supported（需 RPI + 間接固位體）
  • Class III = tooth-supported（圓臂卡環、通常不需間接固位體）
  • 判斷流程：處理第三大臼齒 → 找最遠端 → 判斷游離/支持 → 數 Mod. 區
  • 前牙全缺 + 後牙也缺 → 不是 Class IV！看最遠端
  • 高頻考古題：108-2 #28 / 110-2 #28 / 111-2 #34 / 113-2 #29
CHAPTER 3

活動假牙的骨架

RPD 的三大要素、卡環、RPI 系統
師
阿明，你的嘴巴分類結果是 Kennedy Class II, Mod. 1。右下後牙缺兩顆（游離端），左上中間也缺一顆。我們來設計活動假牙的「骨架」。
你
骨架？活動假牙不就是一塊粉紅色的東西嗎？
師
那只是表面！裡面其實有精密的金屬骨架。設計骨架的核心就是三個字：支撐、固位、穩定。

RPD 三要素

比喻：想像一把椅子
支撐（Support）= 椅腳 → 防止你往下坐穿（義齒防沉陷）
固位（Retention）= 椅子的扶手 → 防止你被往上抬走（義齒防脫落）
穩定（Stability）= 椅背 → 防止你往旁邊倒（義齒防位移）
🖼️

RPI 系統 — 游離端的最佳設計

師
阿明右邊是游離端缺牙（Class II），後面沒有牙齒可以靠。這時候假牙咬東西會往黏膜方向沉下去，如果設計不好，支柱牙會被往後扯壞。所以我要用一套特別的系統：RPI。
R
Rest
放在支柱牙的近心窩
P
Proximal Plate

I
I-bar Clasp

為什麼 Rest 要放近心？

假牙受力會以支點線為軸旋轉下沉。如果 Rest 放在遠心，旋轉的支點就直接壓在支柱牙上面，產生遠心扭力 → 支柱牙被「扳」壞！

放在近心，力量的傳導路徑改善，應力分散到更多結構，支柱牙安全很多。

卡環設計原則 — 倒凹深度與合金選擇

你
卡環要伸入倒凹，但伸多深？不同金屬有差嗎？
師
有！不同材質的卡環能承受的倒凹深度不同。太深會太緊（患者拿不下來），太淺又沒有固位力。
卡環材質適用倒凹深度特點
Cr-Co 鑄造0.25 mm最硬、最不彈 → 只能進入淺倒凹
Gold 鑄造0.50 mm中等彈性
Wrought wire（鍛造線）0.75 mm最有彈性 → 可進入最深倒凹
倒凹深度記憶法
Cr-Co : Gold : Wire = 0.25 : 0.50 : 0.75
（鑄：鑄金：鍛 = 1 : 2 : 3 倍的 0.25mm）

Altered Cast Technique（變更鑄模技術）

你
游離端的黏膜是軟的，牙齒是硬的，同一副印模能同時精確記錄嗎？
師
好問題！這就是 Altered Cast Technique 的用途。先用一般印模取得有牙區域的模型，做好金屬骨架後，再用骨架上的游離端基托做第二次功能性印模，只取游離端黏膜的精確形態，然後把這段模型切掉重新灌模「接」上去。
Altered Cast 適用：Kennedy Class I、Class II（游離端）
目的：讓游離端基托在功能性壓力下取模，確保義齒受力時均勻坐在黏膜上
原理：牙齒是硬的（不可壓縮），黏膜是軟的（可壓縮）→ 兩者需要分別取模

Stress Breaker（應力中斷裝置）

在義齒的支柱牙和游離端基托之間，設計一個可以微動的連接裝置，讓咬合力不直接傳到支柱牙。原理跟 Non-rigid connector 類似，但用於 RPD。

⚠ 考試注意
Stress breaker 的概念已較少使用，現代觀點認為 RPI 系統本身就是最好的「應力中斷」設計 — mesial rest + 短板 + I-bar 的組合已經能有效分散應力。

間接固位體（Indirect Retainer）

你
假牙後面沉下去，前面不就翹起來了嗎？
師
正是！所以我們需要一個東西壓住前面，防止翹起。這就是間接固位體。想像翹翹板，一端下沉，你需要在另一端坐一個人壓住。而且那個人要坐得越遠越好，力臂越長，壓的力量越大。
⚠ 國考超高頻陷阱
「間接固位體距支點線越近越好」→ 完全錯誤！

全口義齒 — 從印模到排牙
師
今天來了另一位患者，王奶奶，80 歲，全口無牙。RPD 需要有牙齒可以勾，但王奶奶一顆都沒有了。我們要做的是全口義齒（Complete Denture）。

沒有牙齒可以卡，全口義齒靠什麼固定？
師
靠三股力量：吸附力（唾液薄膜的表面張力）、邊緣封閉（義齒邊緣的密封效果）、和周圍肌肉的平衡。所以每一步都要非常精確。

初步印模 → 用 Alginate 快速取概形 → 做出個別托盤（Custom tray）

咬合垂直高度（VD）

你
什麼是 VD？為什麼這麼重要？
師
VD 就是上下顎之間的「高度」。想像你的臉是一台手風琴，VD 就是手風琴打開的程度。太高，嘴巴關不攏、肌肉痠痛；太低，嘴巴塌陷、說話含糊。

Neutral Zone（中性區）

你
排牙的位置怎麼決定？
師
人工牙要排在 Neutral Zone（中性區）。舌頭從裡面往外推，嘴唇和頰肌從外面往裡壓，兩股力量平衡的地方。排在這裡，肌肉不會推翻義齒。
Neutral Zone：舌肌向外 = 頰/唇肌向內 → 平衡區
Piezography：利用發音或吞嚥動作，用流動性材料記錄 neutral zone 的技術

排牙原則與平衡咬合

你
天然牙不是追求 MPO（犬齒保護咬合）嗎？全口義齒為什麼不同？
師
因為全口義齒底下沒有牙根抓著骨頭，它就放在黏膜上面。如果側方運動時只有一邊碰到（像 MPO 那樣），另一邊就翹起來了！所以全口義齒要平衡咬合 — 在所有咬合動作中，兩邊同時有接觸，義齒才穩。

Ante's Law — 牙橋設計的根基

你
怎麼知道兩邊的牙齒撐不撐得住這座橋？
師
用 Ante's Law：支柱牙的牙周膜面積加起來，要大於或等於被取代牙齒的牙周膜面積加起來。
Ante's Law 計算範例：
缺 #5（牙根面積 179 mm²）→ 需要 #4（195 mm²）+ #6（431 mm²）= 626 mm²
626 > 179 → 合格，可以做三單位牙橋

但如果支柱牙有 30% 骨喪失 → 有效面積只剩 70%：626 × 0.7 = 438 mm²
仍然 > 179 → 仍可做，但要注意牙周維護

不一定。有幾種設計。

FPD 失敗模式

你
牙橋最常見的失敗原因是什麼？斷裂嗎？
師
不是！最常見的失敗原因是次發性齲齒（secondary caries）。牙冠邊緣密合度不好或口腔衛生不佳 → 蛀牙從邊緣鑽進去 → 支柱牙壞掉。
排名失敗原因預防
1Secondary caries（次發性齲齒）邊緣密合、口腔衛生、定期回診
2牙髓壞死（Pulp necrosis）預備時注意冷卻、保留足夠齒質
3Cement 失敗 / 脫落正確固位設計、適當 cement 選擇
4陶瓷破裂（Porcelain fracture）足夠 connector 尺寸、適當咬合
5牙周病惡化術前評估 + 長期牙周維護

Maryland Bridge（Resin-Bonded FPD）

你
有沒有不用磨那麼多牙齒的牙橋？
師
有！Maryland Bridge 只需要在支柱牙的舌面做極少量的磨牙，用金屬翼片黏在後面。保守但固位力有限。
項目Maryland Bridge（Resin-Bonded FPD）
適應症前牙單顆缺牙、年輕人牙髓腔大不宜大量磨牙
預備方式只磨支柱牙舌面（minimal prep）
固位方式金屬或氧化鋯翼片，用 Resin Cement 黏合

設計原則翼片不能超過接近線 → 否則 path of insertion 困難
Maryland bridge 失敗最常見原因 = Debonding（脫黏）
解決方法：金屬表面做 electrochemical etching 或 sandblasting 增加粗糙度

Ferrule Effect（箍效應）

你
做過根管的牙齒要打釘子（post），上面再做牙冠。有什麼要特別注意的？
師
最重要的一個觀念就是 Ferrule Effect。牙冠的邊緣要能抱住至少 1.5-2mm 的健康牙齒齒質（在 finish line 上方），像一個箍圈一樣把牙齒箍住。如果沒有 ferrule，post-core 受力時牙根容易裂開。
Ferrule Effect：牙冠邊緣上方需有 ≥ 1.5-2mm 的 sound dentin
Ferrule 不足 → 考慮 Crown Lengthening（牙冠延長術） 或 Orthodontic Extrusion（矯正拉出）
沒有 ferrule 的 post-core crown → 根裂風險極高

Provisional Restoration（臨時修復體）

你
牙齒磨好到做好正式牙冠之間，要做臨時冠嗎？
師
一定要！臨時冠有四個重要功能。
功能說明
牙髓保護覆蓋暴露的牙本質 → 防止溫度刺激和細菌入侵
邊緣完整維持 finish line 周圍牙齦健康 → 正式取模時牙齦不腫
咬合穩定防止對咬牙過度萌發、鄰牙傾斜
美觀+功能前牙區維持美觀，後牙區維持咀嚼
⚠ 臨時冠的陷阱
臨時冠如果用 ZOE cement 黏著 → 之後要換 Resin cement 永久黏著前，必須徹底清除 ZOE！
建議換永久冠前 7-10 天回診，改用非 ZOE 類臨時黏合劑。

你
中間那顆假牙（橋體）底部跟牙肉的關係是什麼？

黏合劑（Cement）

種類特點固位力排名注意
Zinc Phosphate傳統首選，歷史最久中高酸性強，刺激牙髓；機械性固位
GIC釋放氟，化學鍵結牙齒中溶解性較高、強度較低
RMGICGIC + 樹脂，強度提升中高操作較 GIC 方便
Resin Cement固位力最強，不溶於唾液最高全瓷冠必用；移除困難
ZOE舒緩牙髓，臨時黏著最低丁香油抑制 Resin 聚合！
Luting vs Bonding 的區別：
Luting（黏合）：Zinc Phosphate、GIC → 靠機械鎖定（mechanical retention）
Bonding（鍵結）：Resin Cement → 靠化學鍵結 + 微機械鎖定 → 固位力最強
全瓷冠因為不能做固位溝（脆性材料），所以必須用 bonding（Resin cement）
⚠ 重要陷阱
臨時冠如果用了 ZOE cement → 換永久 Resin cement 之前，必須徹底清除 ZOE 殘留，否則 Resin 聚合不完全！

Wetting（潤濕性）& Contact Angle（接觸角）

你
灌石膏的時候，有的印模材很容易灌、有的一直有氣泡，為什麼？
師
關鍵是 Contact Angle（接觸角）。接觸角越小，材料越「親水」，石膏漿越容易流入細節。
材料親水性Contact Angle灌模品質

Addition Silicone本質疏水中需加 surfactant 改善
Polysulfide中中尚可

Contact angle 小 = 親水 = wetting 好 = 灌出來的模型細節好。
Addition Silicone 本身是疏水性材料，但現代產品加了 surfactant → 變成「親水性加聚矽膠」。

Agar-Alginate 聯合印模（Laminate Technique）

你
可逆膠體（Agar）不是已經淘汰了嗎？
師
單獨用確實少了，但 Agar-Alginate Laminate Technique 還會考。原理是：用精確的 Agar 做齒質面的精密印模，外面包 Alginate 當支架 — 兩者是同類膠體，接面可以化學結合。
Agar（可逆膠體）：加熱液化 → 冷卻固化 → 可重複使用
Alginate（不可逆膠體）：化學反應固化 → 不可逆
Laminate 聯合印模：Agar（精密層）+ Alginate（外層）→ 兩者都是 hydrocolloid → 可化學鍵結

Gypsum Products（石膏製品）

你
石膏不就是灌模用的白色粉末嗎？還分這麼多型？

石膏有五個 Type，強度和膨脹率不同，用途天差地別。搞混了，做出來的假牙就不合。

考試常問「哪個膨脹最小」→ 答案是 Type IV。

Porcelain Firing Sequence（燒瓷順序）

你
PFM 的瓷是一次燒上去的嗎？
師
不是！要分層燒，每一層有不同功能。順序很重要，考試愛考。
Opaque layer（遮色層）— 遮蓋金屬色、建立瓷金結合（最關鍵的一層！）
Body / Dentin porcelain（牙本質層）— 提供主色調和體積
Incisal / Enamel porcelain（切端/釉質層）— 提供透明感
Glaze（釉燒）— 表面光滑、封閉氣孔、增加美觀
每次燒瓷溫度依序降低（Opaque 最高 → Glaze 最低）
→ 避免後燒的高溫破壞前面已燒好的層次。
Opaque 層最重要：決定瓷金結合強度 + 遮色效果。

PFM 熱膨脹係數（CTE）匹配

你
金屬和瓷的熱膨脹不一樣，不會裂開嗎？
師
問得好！關鍵是金屬的 CTE 必須略高於陶瓷。這樣冷卻時金屬收縮多一點，把陶瓷「抱緊」，陶瓷處於壓縮狀態 → 壓縮力讓陶瓷更強！
CTE 規則（必背！）：
Metal CTE 略高於 Porcelain CTE → 冷卻時金屬收縮稍多 → 陶瓷受壓縮應力 → 增強
如果 Metal CTE < Porcelain CTE → 陶瓷受張力 → 裂瓷！
如果兩者差太大 → 也會裂（應力過大）
「略高」才是完美匹配。

Adhesive vs Cohesive Failure（黏著失敗 vs 內聚失敗）

失敗類型斷裂位置臨床意義
Adhesive failure在界面斷裂（瓷和金屬之間）結合力不足 → 可能是 opaque 層問題、表面汙染、CTE 不匹配
Cohesive failure在材料本身內部斷裂結合力強於材料本身 → 理想黏著（但可能材料太薄或受過大外力）
Cohesive failure 反而代表結合良好 — 因為黏合界面比材料本身更強。
PFM 的裂瓷如果是 cohesive failure（瓷本身碎裂）→ 瓷太薄或咬合力太大。
如果是 adhesive failure（金屬面乾淨露出）→ 瓷金結合有問題。

Law of Beams（梁定律）

你
老師，FPD pontic 為什麼太長會斷？有數學公式嗎？
師
有！就是 Law of Beams。這條公式告訴你，長度影響最大，而且是三次方！
Deflection ∝ L³ / (W × T³)

L = 長度（Pontic span）→ 影響最大！長度加倍 → 撓度變 8 倍
W = 寬度 → 加寬可減少撓度（線性關係）
T = 厚度 → 影響第二大！厚度加倍 → 撓度變 1/8

臨床意義：Pontic 不能太長、connector 要夠厚 → 寧可增加 abutment 也不要做太長的橋體。

鑄造三大膨脹（補償蠟型收縮）

你
蠟型燒掉之後，金屬冷卻會收縮，怎麼補償？
師
靠包埋材的三種膨脹來補償！每一種的控制方式不同。
膨脹類型原理控制方式
Setting expansion（硬化膨脹）包埋材硬化時結晶膨脹W/P ratio（水粉比）
Hygroscopic expansion（吸水膨脹）硬化過程中接觸水 → 額外膨脹水浴法 → Gypsum-bonded 專用
Thermal expansion（熱膨脹）加熱燒蠟時受熱膨脹最終加熱溫度
Hygroscopic expansion 只用於 Gypsum-bonded investment（因為石膏結合型的 thermal expansion 不夠大）。
Phosphate-bonded investment 靠 thermal expansion 就足夠補償，不用吸水膨脹。
Ringless technique：不用金屬鑄造環 → 膨脹量更大（不受環限制）→ 適合高熔點合金。

  • Polyether：親水性最佳但不能泡水（吸水膨脹）；contact angle 最小
  • Polysulfide：working time 最長、味道最臭、尺寸穩定性最差
  • Agar-Alginate laminate technique：同為膠體可化學結合
  • 石膏 Type IV（Die stone）：做 die 最常用、膨脹率最低；Type V 膨脹反而較高
  • 合金分類：High noble = noble ≥60% 且 Au ≥40%；Noble = noble ≥25%
  • Zirconia 強度高的原因：應力誘發相轉換（tetragonal → monoclinic → 體積膨脹壓裂縫）
  • PFM 燒瓷順序：Opaque → Body → Incisal → Glaze（溫度遞減）
  • PFM CTE 規則：Metal CTE 略高於 Porcelain CTE → 陶瓷受壓縮 → 增強
  • Law of Beams：Deflection ∝ L³ / (W × T³) → 長度影響最大
  • Adhesive failure = 界面斷；Cohesive failure = 材料內斷（反而代表結合良好）
  • Gypsum-bonded → 金合金 + hygroscopic expansion；Phosphate-bonded → PFM + thermal expansion

咬合的平衡

MPO、Posselt、Bennett、咬合器、Hanau's Quint、咬合調整
你
前面提到天然牙要 MPO、全口義齒要平衡咬合，到底差在哪裡？
師
這是整個咬合學的核心，讓我好好解釋。

MPO（Mutually Protected Occlusion）

MPO 又叫犬齒保護咬合，是天然牙的黃金標準。核心邏輯是「互相保護」：

比喻：日夜班保全
正中咬合（CO）時：後牙接觸、前牙不碰 → 後牙當「日班保全」，保護前牙不被垂直力磨耗
側方/前突運動時：前牙（犬齒）引導、後牙全分離 → 犬齒當「夜班保全」，保護後牙不受側向力破壞
🖼
MPO 互相保護咬合示意
CO 時後牙接觸、前牙不碰；側方運動時只有犬齒引導、後牙全分離

MPO 四大條件

#條件
1CR = CO（正中關係 = 正中咬合）
2CO 時：後牙接觸、前牙不接觸
3側方及前突運動時：只有犬齒接觸，後牙全部分離
4牙周健康、Angle Class I
⚠ 常見陷阱
「側方運動時，工作側後牙有接觸」→ 這是 Group Function（群功能咬合），不是 MPO！
MPO = 只有犬齒引導 → 工作側和非工作側後牙都不接觸。

MPO 不適用的三種情況

Angle Class II / III前牙關係異常，犬齒無法引導
全口義齒改用平衡咬合
嚴重磨牙症犬齒磨耗，引導功能喪失 → 改用 Group Function

Functional Cusp vs Non-Functional Cusp（功能尖 vs 非功能尖）

你
哪個是「功能尖」？上顎頰尖還是舌尖？
師
功能尖 = Centric holding cusp = 咬在對咬牙窩裡的那個尖。記住：上顎舌尖和下顎頰尖是功能尖。
功能尖（Centric holding cusp）非功能尖（Non-functional cusp）
上顎舌尖（Palatal cusp）頰尖（Buccal cusp）
下顎頰尖（Buccal cusp）舌尖（Lingual cusp）
功能尖咬在對咬窩裡，維持垂直高度（VD）→ 不能隨便磨！
咬合調整時，原則上磨非功能尖來消除干擾。

Long Centric（Freedom in Centric）

你
MPO 要求 CR = CO，但有時候不完全一致？
師
對！Long centric 就是允許從 CR 到 CO 之間有 0.5-1mm 的前後方向自由空間。下顎可以在這個範圍內自由滑動，不會有干擾。
Long centric = Freedom in centric
CR 和 CO 之間允許 0.5-1mm 的前後方向（anteroposterior）自由空間
目的：容納下顎自然的微小前後移動，減少咬合干擾
注意：只允許前後方向的自由，不允許側方自由

Posselt's Envelope of Motion（下顎運動軌跡）

你
下顎到底能怎麼動？有沒有一張圖可以看全部？
師
有！就是 Posselt's Envelope。從側面（矢狀面）看，下顎所有可能的運動軌跡形成一個菱形的範圍。
Posselt's Envelope 矢狀面重要地標（由上到下、由後到前）：
1. CR（Centric Relation） → 最上後方的點（髁突在關節窩最上前內位）
2. CO / MIP（最大嵌合位） → CR 前方約 1mm
3. 最前突位（Maximum protrusion） → 下顎最前方能到的點
4. 最大張口位 → 最下方的點

CR 是 border movement 的起點（可重複性最高的參考位置）。
CO 不在邊界上，而是在 envelope 內部的功能區。

Bennett Movement（班尼特運動）

你
側方運動時，兩邊的髁突分別怎麼動？
師
工作側（orbiting side）的髁突原地轉動，非工作側（balancing side）的髁突向前、內、下移動。這個非工作側髁突的移動軌跡和矢狀面的夾角，就叫 Bennett angle。
類型說明臨床影響
Immediate side shift
（即刻側移）		側方運動開始瞬間，整個下顎先做一個純側向平移（無旋轉）越大 → 義齒需要更多 freedom in centric
Progressive side shift
（漸進側移）		之後非工作側髁突沿斜面漸進地向前內下移動由 Bennett angle 的大小決定
Bennett angle = 非工作側髁突運動軌跡與矢狀面的夾角
→ 反映關節窩內側壁的斜度
→ 用於調整 semi-adjustable articulator
Bennett angle 越大 → 後牙咬合面形態需更大的 freedom

Working Side vs Non-Working Side Interference

你
側方運動時，工作側和非工作側的干擾哪個比較危險？
師
非工作側（Balancing side）干擾更危險！因為它離支點（工作側髁突）最遠，槓桿效應最大，對 TMJ 和牙周的破壞力最強。
⚠ 必背
Non-working side（balancing side）interference 比 working side interference 更具破壞性。
原因：距離工作側髁突越遠，力臂越長，對 TMJ 施加的扭矩越大。
→ 天然牙製作修復體時，一定要檢查並消除非工作側干擾。

Christensen's Phenomenon

你
全口義齒患者做前突運動時，後面會怎樣？
師
後牙會張開！這叫 Christensen's phenomenon — 前突運動時，髁突沿關節粗隆下滑，後方產生楔形間隙（posterior open bite）。全口義齒如果沒有平衡咬合來填補這個間隙，義齒就會翹起來。
Christensen's phenomenon：protrusive movement 時後牙區出現的楔形間隙（posterior open bite）
→ 全口義齒需要補償曲線（compensating curve）和足夠的牙尖角度來消除此間隙
→ 這就是為什麼 Hanau's Quint 如此重要

這是控制全口義齒平衡咬合的五個關鍵旋鈕：

Hanau's Quint 互動關係

如果...則需要...原因

簡單記：Condylar guidance 和 Incisal guidance 越陡 → 後三個因素（Cusp、Curve、Plane）需要越大來補償。
它們之間是正比關係（一個增加，另一個也要增加才能維持平衡）。

Buccal Upper · Lingual Lower

關節的煩惱

明
陳醫師，我最近張嘴有喀喀聲，有時候嘴巴張不開，是怎麼了？
師
阿明，這可能是顳顎關節紊亂症（TMD）。讓我檢查一下。

TMD 四大類型

類型特徵關鍵鑑別
肌肉性（Myofascial pain）咀嚼肌痠痛、壓痛沒有關節聲音
關節盤前移位 WITH reduction張口時 Click（彈響）Click 後功能正常（盤子有回位）
關節盤前移位 WITHOUT reduction嘴巴張不開（< 40mm）「閉鎖」（Closed lock），沒有 click

Click（彈響）= 關節盤快速移位的聲音 → With reduction
Crepitus（摩擦音）= 骨對骨摩擦 → Osteoarthritis 或盤穿孔

Myofascial Pain 診斷要點

你
肌肉性 TMD 跟關節問題怎麼區分？
師
肌肉性 TMD 的關鍵是 trigger point（激痛點）和referred pain（轉移痛）。你按壓某個肌肉的激痛點，疼痛會放射到遠處 — 例如按壓 masseter 的 trigger point，患者說太陽穴痛。
Myofascial pain 診斷標準：
1. 咀嚼肌區域有鈍痛（dull, aching pain）
2. 觸診有 trigger point（激痛點）— 按壓會引發疼痛
3. Referred pain（轉移痛）— 痛感轉移到其他區域
4. 關節本身通常無聲音、無結構變化

Wilkes Staging System（內部紊亂分期）

你
關節盤的問題有嚴重程度之分嗎？
師
有！Wilkes 把 internal derangement 分成五期，從輕微的 click 到嚴重的關節退化。
Stage臨床表現影像
I（Early）偶爾 click、無痛盤稍微前移、形態正常
II（Early-intermediate）頻繁 click、偶爾疼痛、偶爾張口受限盤前移、形態開始變化
III（Intermediate）頻繁疼痛、intermittent locking盤前移、盤變形
IV（Intermediate-late）持續疼痛、chronic closed lock盤前移+變形、早期骨變化
V（Late）crepitus、可能減痛（燒壞了）盤穿孔、嚴重骨變化（OA）
Wilkes I-II：有 click、盤可復位 → 保守治療
Wilkes III：間歇性閉鎖 → 考慮 anterior repositioning splint
Wilkes IV-V：慢性閉鎖或退化 → 可能需要手術介入

咬合板（Occlusal Splint）類型

你
咬合板不就是 night guard 嗎？還有別的？
師
有兩大類，功能完全不同！
類型設計適應症注意事項
Stabilization splint
（穩定型咬合板）		Full-arch、flat plane（全弓、平面）Myofascial pain、Bruxism、一般 TMD最常用、最安全；不改變咬合關係
Anterior repositioning splint
（前位化咬合板）		有斜面引導下顎向前Disc displacement with reduction（Click）可能造成咬合改變（後牙 open bite）；只短期使用
⚠ Bruxism 的咬合板選擇
磨牙症（Bruxism）用 stabilization splint，不用 anterior repositioning splint！
原因：Bruxism 是中樞性問題（不是咬合問題），不需要重新定位下顎。
Stabilization splint 的平面設計可以分散咬合力、保護牙齒。
Bruxism 的病因是中樞性（central），不是周邊性（peripheral）。
→ 調整咬合不能治癒磨牙
→ 治療：stabilization splint（保護牙齒）+ 行為治療 + 必要時藥物

鑑別診斷

你
哪些疾病的症狀跟 TMD 很像？
師
好幾個！漏診的話後果很嚴重，特別是心臟轉移痛。
鑑別診斷與 TMD 的區別
Trigeminal neuralgia
（三叉神經痛）		電擊樣劇痛、持續數秒、有 trigger zone（如觸摸臉部某區域）；TMD 是鈍痛

（莖突過長症候群）		莖突過長壓迫周圍組織；吞嚥或轉頭時痛；觸診 tonsillar fossa 可誘發

（心臟轉移痛）		心絞痛或心肌梗塞可轉移至左側下顎；伴隨胸悶、呼吸困難 → 緊急！
Otitis media
（中耳炎）		耳痛為主、可能有發燒和聽力下降
Temporal arteritis
（顳動脈炎）		顳部頭痛、ESR 升高、年齡 >50 歲為主

  • Crepitus = 骨擦音 = Osteoarthritis 或盤穿孔
  • 正常張口 ≥ 40mm
  • Myofascial pain：trigger point + referred pain + 無關節聲音
  • Wilkes staging：I-II 保守治療、III intermittent locking、IV-V 可能需手術

  • MRI = 看關節盤的 gold standard；CT = 看骨頭
  • TMJ ankylosis：True（骨融合）vs Pseudo（關節外原因）
  • 治療順序：保守 → 積極（衛教 → 咬合板 → 物理 → 藥物 → 注射 → 手術）

Biological Width

比喻：牙齒的「安全距離」
想像牙齒是一棟房子，牙齦是圍牆。圍牆跟房子之間有一段「不能動」的安全距離 — 這就是 Biological Width。如果你的修復體邊緣侵入了這個距離，圍牆就會開始崩壞（牙周發炎、骨吸收）。

Gingival Biotype（牙齦生物型）

你
同樣侵犯 BW，有些患者反應很大、有些卻還好？
師
因為牙齦有不同的生物型！薄的跟厚的反應完全不同。
特徵Thin Scalloped（薄貝殼型）Thick Flat（厚平坦型）
牙齦厚度薄（< 1mm）厚（≥ 1mm）
齦緣形態高低起伏大（scalloped）平坦
骨形態骨開窗/裂開風險高骨緻密、較穩定
侵犯 BW 後牙齦萎縮（recession）骨吸收 + 牙周袋
Margin 位置建議盡量 supragingival（齦溝上方）可接受 subgingival
Thin biotype 侵犯 BW → recession（牙齦退縮）
Thick biotype 侵犯 BW → 骨吸收形成牙周袋
臨床選邊緣位置時，thin biotype 患者要特別保守 → 邊緣留在齦溝上方。

Subgingival Margin（齦下邊緣）何時使用？

你
邊緣不是放越下面越美觀嗎？
師
錯！邊緣放在齦溝下方風險很高。只有三種情況才考慮 subgingival margin。
Subgingival margin 的三個適應症（只有這三種才需要！）：
1. 齲齒延伸到齦溝下
2. 既有修復體的邊緣已在齦溝下
3. 美觀需求（前牙區需隱藏金屬邊緣或不透明邊緣）

其他情況 → 盡量用 supragingival 或 equigingival margin

Crown Lengthening（牙冠延長術）

如果修復體邊緣需要往下延伸，但空間不夠，就要做牙冠延長術：

手術類型等待時間規則
有骨切除（Osseous surgery）3-6 個月修復體邊緣必須離 bone crest ≥ 3mm
只切軟組織（Gingivectomy）3 個月前提是 attached gingiva 足夠
Crown lengthening 的 3mm rule：
修復體邊緣到 alveolar bone crest 至少要 3mm（= BW 2mm + sulcus 1mm）
如果不夠 → 做骨切除把骨頭往下移，製造足夠空間。

Translucency & Metamerism

你
為什麼有些假牙在診間看起來很搭，到外面卻變色？
師
那叫 Metamerism（同色異譜）！兩個顏色在某種光源下看起來一樣，但換另一種光源就不一樣了。
Metamerism：兩個物體在一種光源下顏色匹配，但在不同光源下不匹配。
原因：光譜反射曲線不同，只在特定光源下「巧合」看起來一樣。
→ 比色時最好用多種光源確認（自然光 + 診間燈）

Translucency（透明度）：天然牙切端較透明、頸部較不透明（opacity）。
全瓷冠比 PFM 透明度更好（PFM 有金屬遮光 → 需加 opaque layer → 透明感差）。

Porcelain Laminate Veneer（陶瓷貼面）

你
貼面就是很薄的陶瓷片貼上去？
師
對！最大的優點是修磨量極少，保留最多齒質。但有前提條件。
項目傳統 VeneerMinimal / No-prep Veneer
修磨量0.3-0.5mm（labial surface）0-0.3mm（contact lens veneer）
邊界位置盡量停在釉質內完全在釉質內
黏合劑必須用 Resin Cement（light-cure 或 dual-cure）
適用變色、形態不佳、輕微排列不齊主要改善形態，顏色變化不大
Veneer 成功的關鍵：bonding 在釉質上
釉質的 etch + bonding 效果遠優於牙本質 → 邊界留在釉質內 = 黏著力最好
如果修磨到牙本質過多 → 改做全瓷冠（bonding 需求不同）
⚠ Veneer 禁忌
1. 嚴重磨牙症（Bruxism）→ 咬合力過大會把貼面咬裂
2. Deep bite 且 edge-to-edge 咬合 → 切端直接承受力量
3. 大面積牙本質暴露（bonding 效果差）

植體美學（Implant Aesthetics）

你
植牙的美觀跟假牙有什麼不同？
師
植牙沒有牙周韌帶，軟組織行為跟天然牙不一樣。要做出漂亮的植牙，有兩個關鍵概念。
概念說明臨床意義
Platform switchingAbutment 直徑小於植體平台直徑將 micro-gap 內移 → 減少 crestal bone loss → 保存齦乳頭高度
Emergence profile
（穿出輪廓）		從植體平台到牙冠邊緣的輪廓形態需模擬天然牙根的漸變形態 → 支撐軟組織 → 看起來自然
Platform switching 的好處：
1. 保存 crestal bone（嵴頂骨）→ 維持牙齦高度
2. 減少 micro-gap 附近的發炎反應
3. 有利於 papilla preservation（齦乳頭保存）

Emergence profile 要從 provisional restoration 開始塑形 → 軟組織會依此形態癒合。
