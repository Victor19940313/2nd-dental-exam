CHAPTER 1

矯正的語言

Angle 分類、Andrew's Keys、骨骼關係與臉型
師
小安，我們先來看你的咬合。矯正醫師的第一步就是「分類」，就像急診要先分檢傷等級一樣。我們用的是 Angle Classification。
你
Angle 分類只看大臼齒的關係對吧？
師
沒錯，看的是上顎第一大臼齒的 mesiobuccal cusp 跟 下顎第一大臼齒的 buccal groove 的關係。

Angle 分類

Angle 三級分類：Class I（正常大臼齒關係）、Class II（下顎後縮）、Class III（下顎前突）
分類大臼齒關係特徵盛行率
Class I上6 MB cusp 對齊下6 buccal groove大臼齒關係正常，但可能有擁擠、間隙、開咬等問題~50-55%
Class II div.1下顎後縮（distal）至少半顆 cusp上門牙前傾（proclined）、OJ 增加、常有 deep bite~15%
Class II div.2下顎後縮至少半顆 cusp上門牙後傾（retroclined）、OJ 正常或減少、deep bite 更嚴重~3-5%
Class III下顎前移（mesial）至少半顆 cusp下顎前突、anterior crossbite、negative OJ~5% (亞洲人較高 ~15%)
生活化比喻
想像兩排齒輪要互相咬合：
Class I = 齒輪完美嚙合
Class II = 下排齒輪往後退了，上排突出來 → 暴牙
Class III = 下排齒輪往前衝了 → 戽斗
陷阱題
Angle 分類只描述大臼齒的牙齒關係，不等於骨骼關係！一個 Class I molar 的病人可以有 skeletal Class II 或 Class III。判斷骨骼關係要用 cephalometric analysis（ANB、Wits）。

Andrew's 六大咬合關鍵（Six Keys to Normal Occlusion）

你
除了 Angle 分類之外，怎麼定義「正常咬合」？
師
Andrews 在 1972 年研究了 120 副正常咬合模型，歸納出六大關鍵。這也是 straight-wire bracket 設計的理論基礎。
Key名稱內容
Key 1Molar relationship上6 MB cusp 對齊下6 buccal groove；上6 distal marginal ridge 對齊下6 mesial marginal ridge
Key 2Crown angulation (tip)每顆牙的牙冠 gingival 部分比 incisal 部分更 distal（正角度）= mesiodistal 傾斜
Key 3Crown inclination (torque)上前牙牙冠唇側傾斜（正 torque），上後牙牙冠舌側傾斜（負 torque），下後牙也是舌側傾斜
Key 4No rotations沒有旋轉。旋轉的牙齒佔空間、影響咬合
Key 5Tight contacts沒有間隙（diastema），接觸點緊密
Key 6Curve of Spee平坦到輕微（0-2 mm），不超過 1.5 mm 為理想
記憶口訣：Andrew's 六把鑰匙
莫（Molar）角（Angulation）傾（Inclination）轉（Rotation）隙（Spacing）彎（Curve）

臉型分類

三種臉型：Mesocephalic（正常）、Dolichocephalic（長臉高角）、Brachycephalic（短臉低角）
師
小安的臉型偏長，我們叫 dolichocephalic（長頭型）。不同臉型對矯正治療的影響很大，特別是在 vertical dimension 的控制上。
臉型別名MPADivergence特徵
Dolichocephalic長臉型 / hyperdivergent>30°高角下顎平面角大、前開咬傾向、後旋轉、弱咬肌
Mesocephalic正常臉型 / normodivergent25-30°平均標準比例，治療預後最佳
Brachycephalic短臉型 / hypodivergent<25°低角下顎平面角小、deep bite、前旋轉、強咬肌
陷阱題
高角（dolichocephalic）患者要避免 extrusion，因為會讓臉更長、開咬更嚴重。治療策略是intrusion + 控制 vertical dimension。
低角（brachycephalic）患者咬肌力量強，anchorage 消耗快，要特別注意錨定控制。
你
MPA 就是 mandibular plane angle 嗎？
師
對，就是 SN 平面或 FH 平面跟下顎平面（Go-Me）的夾角。FMA（Frankfort Mandibular Angle）正常值約 25°，SN-MP 正常值約 32°。高角超過這個值，低角低於這個值。

診斷基礎考試重點

考試重點框
  • Angle Class I：上6 MB cusp 對齊下6 buccal groove（正常大臼齒關係）
  • Class II div.1：上門牙 proclined，OJ 增加；div.2：上門牙 retroclined，deep bite 嚴重
  • Class III：下顎前移至少半顆 cusp，亞洲人盛行率較高（~15%）
  • ANB = 2°（正常），>4° = skeletal Class II，<0° = skeletal Class III
  • Wits appraisal = -1 mm：不受 cranial base 影響，比 ANB 可靠
  • Andrew's 六鍵：molar relationship、angulation、inclination、rotation、spacing、Curve of Spee
  • Protrude/retrude = 位移；Procline/retrocline = 傾斜
  • OJ = 2-4 mm（水平），OB = 2-4 mm（垂直，30-50%）
  • 高角 = MPA > 30° → 避免 extrusion；低角 = MPA < 25° → anchorage 消耗快
CHAPTER 2

生長的節奏

Scammon 曲線、骨化模式與顎骨生長
你
林醫師，小安現在 15 歲，她還在生長嗎？矯正治療的時機跟生長有關係嗎？
師
關係非常大！矯正很多時候是在「借力使力」，利用病人還在長的時候來導引顎骨生長。先來搞清楚生長（growth）和發育（development）的差別。

生長 vs 發育 + Scammon 曲線

Growth（生長）= 大小的增加（量的改變），可以測量
Development（發育）= 成熟度的改變（質的改變），功能的進步
Maturation（成熟）= 達到成人狀態的過程
師
Scammon 在 1930 年發表了著名的生長曲線，把身體各系統的生長模式分成四種。國考超愛考。
系統曲線型態關鍵時間點例子
Neural（神經型）早期快速上升6-7 歲達成人的 ~90%，12 歲 ~100%腦部、眼睛、頭顱
Lymphoid（淋巴型）先升後降（倒 U 型）10-12 歲達 200% 高峰，然後退縮到 100%扁桃腺、腺樣體、淋巴結
General（一般型）S 型曲線嬰幼期快 → 兒童期慢 → 青春期第二波加速骨骼、肌肉、內臟、身高
Genital（生殖型）青春期才起飛青春期前幾乎不長，12-18 歲急速上升性腺、性器官
Scammon 四大生長曲線：Neural（最早達100%）、Lymphoid（倒U型超過100%）、General（S型）、Genital（最晚起飛）
陷阱題
「淋巴型組織在什麼年齡達到最大？」→ 10-12 歲，而且會超過成人值（達 200%）！之後才退縮。
「腦部在幾歲達到成人大小的 90%？」→ 6-7 歲（Neural 型）。這就是為什麼頭圍在嬰幼兒期就接近成人。

骨化模式

膜內骨化（上顎、顱頂）vs 軟骨內骨化（下顎髁突、蝶枕聯合），各部位骨化方式一覽
師
顎骨的生長方式有兩種：膜內骨化（intramembranous）和軟骨內骨化（endochondral）。哪裡用哪種，考試一定要會。
骨化模式機制位置
Intramembranous
（膜內骨化）		間葉組織直接變成骨頭，不經過軟骨階段上顎、顱蓋骨（calvaria）、下顎骨體部、鎖骨
Endochondral
（軟骨內骨化）		先形成軟骨模型 → 軟骨被骨頭取代顱底（cranial base）、下顎髁突（condyle）、長骨

顱底縫合（Synchondroses）

縫合全名閉合時間
SESSpheno-ethmoidal synchondrosis6-7 歲
ISSIntersphenoid synchondrosis出生前已閉合
SOSSpheno-occipital synchondrosis生長期結束才閉合（~13-16 歲女；15-17 歲男）
SOS 最晚閉合的記憶法
SOS = Save Our Ship = 最後才離開船（最晚閉合）
SOS 是國考最常考的 synchondrosis，因為它是最晚閉合的，也是顱底最後的生長中心。SOS 閉合後，顱底長度就固定了。

上下顎生長

下顎生長：髁突向後上方生長，下顎整體向前下方位移；上顎靠縫合生長向前下方移動
下顎旋轉：Forward rotation（短臉傾向）vs Backward rotation（長臉傾向），影響臉型與咬合
你
上顎跟下顎的生長方式一樣嗎？
師
完全不一樣！上顎主要靠膜內骨化 + 縫合生長，下顎主要靠髁突的軟骨內骨化。

上顎（Maxilla）生長

方向機制時間
寬度Midpalatal suture 膜內骨化~10 歲左右完成大部分
前後長度後方骨沈積（tuberosity apposition）~10 歲完成大部分
垂直高度前方 = bone apposition；後方 = 鼻底下降 + alveolar 生長持續到青春期後
上顎整體是向前向下生長（forward and downward），但生長的方式其實是後方和上方做 apposition（骨沉積），前方和下方做 resorption（骨吸收），等於「翻譯」到前下方。

下顎（Mandible）生長

結構機制
Condyle（髁突）軟骨內骨化 → 向後上方生長 → 下顎整體位移到前下方
Ramus前方吸收（resorption）+ 後方沈積（apposition）→ ramus 後退、body 變長
Chin幾乎不生長（remodeling only）
比喻時間
下顎 ramus 的生長就像「倒退走路」：前面的骨頭被挖掉（吸收），後面長出新骨頭（沈積）。結果 ramus 整體「往後退」，但下顎 body 的空間就變長了。

下顎旋轉與臉型

旋轉方向臉型結果特徵
前旋轉 (Forward rotation)短臉型 / deep bite下顎平面角減小、chin 變突出、brachycephalic
後旋轉 (Backward rotation)長臉型 / open bite下顎平面角增大、chin 後縮、dolichocephalic

萌發問題

你
萌發延遲怎麼定義？
師
通常超過正常萌發時間 6 個月到 1 年以上就算延遲。原因很多：局部性（多生牙、牙瘤、纖維化牙齦）或全身性（甲狀腺低下、cleidocranial dysplasia、Down syndrome）。
問題定義關鍵點
Delayed eruption萌發比正常時間延遲先排除局部障礙（多生牙、牙瘤），再考慮全身因素
PFE（Primary failure of eruption）牙齒無法萌發，且無法被矯正力拉出突變基因：PTHR1（PTH1R）。施力拉牙會 ankylose，禁止orthodontic traction
Ankylosis（黏連）牙齒與骨頭直接融合，無 PDL叩診有金屬音（high-pitched）、infraocclusion、不能矯正移動
陷阱題
PFE 跟 ankylosis 不一樣！PFE 是基因突變（PTHR1）導致牙齒「不想萌發」，即使用矯正力拉也沒用而且會造成 ankylosis。而一般 ankylosis 是外傷或感染後「已萌發的牙齒」跟骨頭黏住。

牙弓發育

師
牙弓的寬度和周長在不同年紀會改變。矯正醫師要知道哪些尺寸還會變，哪些已經定型了。
測量變化臨床意義
Intercanine width乳牙列到恆牙早期增加 ~3-4 mm，之後幾乎不再增加矯正不宜過度擴張犬齒間寬度（容易復發）
Intermolar width混合齒列到恆牙列增加 ~1-2 mm擴弓在青春期前效果較好（midpalatal suture 未閉合）
Arch perimeter從混合齒列到恆牙列減少 ~3-4 mm（下顎更明顯）牙弓周長自然會縮小，這是正常現象
Curve of Spee正常 0-4 mm，理想 ≤ 2 mm每 1 mm 的 Curve of Spee 需要約 1 mm 的空間來整平

Leeway Space：乳犬齒+乳臼齒寬度 > 恆犬齒+恆小臼齒寬度的差值（上顎 0.9 mm、下顎 1.7 mm/側）

你
小安的牙齒很擠，空間從哪裡來？
師
先搞清楚一個重要觀念：incisor liability（門齒負債）。

骨頭的回應

力的傳導、骨吸收與牙齒移動生理學
安
林醫師，矯正器的力量是怎麼讓牙齒移動的？不會弄壞骨頭嗎？
師
好問題！牙齒移動的本質就是：施力 → 牙周韌帶（PDL）感受壓力 → 骨頭重塑（remodeling）。壓力側骨吸收，張力側骨沈積。但力量太大反而會適得其反。

力的傳導（Mechanotransduction）

矯正力施加到牙齒 → PDL 受壓 → 細胞感應到機械刺激 → 轉化為生化訊號 → 啟動骨吸收 / 骨沈積
Piezoelectric effect（壓電效應）：骨頭受壓彎曲時產生微小電流。
壓力側（compression side）→ 負電 → 骨吸收（osteoclast 活化）
張力側（tension side）→ 正電 → 骨沈積（osteoblast 活化）
比喻時間
把牙齒想像成插在泥土裡的木樁。你推木樁，前面的泥土被壓縮（壓力側 → 挖掉），後面的泥土被拉開（張力側 → 填土）。結果木樁就移動了。但如果你用力過猛，前面的泥土會塌陷（hyalinization），反而移不動。

輕力 vs 重力反應

輕力：直接骨吸收（frontal resorption），速度快、損傷小；重力：玻璃樣變 → 間接骨吸收（undermining resorption）
你
為什麼矯正教科書一直強調要用「輕力」？
師
因為輕力（light force）產生的是直接骨吸收（frontal resorption），速度快、損傷小。重力會壓死 PDL → 產生玻璃樣變（hyalinization）→ 只能靠間接吸收（undermining resorption），速度慢而且傷牙根。

骨吸收系統

RANK-RANKL-OPG 系統：RANKL 活化破骨細胞促進骨吸收，OPG 為誘餌受體抑制骨吸收
師
骨吸收的分子機轉核心是 RANK-RANKL-OPG 系統。
分子來源功能
RANKLOsteoblast / PDL 細胞表面與 osteoclast 前驅細胞上的 RANK 結合 → 促進骨吸收
RANKOsteoclast 前驅細胞 / 成熟 osteoclast接受 RANKL 訊號 → 分化成成熟 osteoclast
OPGOsteoblast誘餌受體（decoy receptor），攔截 RANKL → 抑制骨吸收
Frontal resorption（直接吸收）：osteoclast 直接從 PDL 側吸收壓力側的骨板。效率高，是理想的移動方式。
Undermining resorption（間接吸收）：PDL 壞死（hyalinization）→ osteoclast 從骨髓腔側向外挖掘。速度慢、不可控、可能造成 EARR。
Hyalinization（玻璃樣變）：PDL 中膠原纖維壞死，呈現透明無結構的外觀 → 阻礙牙齒移動 → 力量去除或減輕後才能恢復。

Strength 不變
Stiffness ↓ 1/8（1/L³）

NiTi（鎳鈦合金）是矯正界的明星材料。它有兩個獨特性質：shape memory（形狀記憶）和 superelasticity（超彈性 / 偽彈性）。

Austenite（沃斯田鐵相）高溫（口腔溫度以上）硬、有彈性、是「記憶形狀」的相態

FA 的理論基礎是 Moss 的 Functional Matrix Theory：「不是骨頭決定功能，而是功能決定骨頭。」改變肌肉、軟組織的功能環境 → 引導骨骼生長方向。但要在生長期才有效。

功能性矯正原理

Functional Matrix Theory（Moss, 1962）：骨骼的生長受到周圍軟組織功能的調控。
FA 透過改變下顎位置（前移或後移）→ 改變肌肉張力和關節受力 → 刺激或抑制骨骼生長

最佳使用時機：生長高峰前 1-3 年（女生 ~9-10 歲，男生 ~11-12 歲）
效果：大部分 FA 的效果是 70% 牙齒性 + 30% 骨骼性（不是純粹改變骨骼）

正顎手術適應症、術式與 Surgery First
你
林醫師，什麼情況下矯正做不到的，要靠正顎手術？
師
當骨骼差異太大，光靠牙齒移動無法 camouflage（偽裝）時，就需要手術介入。國考會考適應症的閾值和術式比較。

適應症

師
以 Class III（下顎前突或上顎後縮）為例，當超過以下數值時，通常建議手術。
指標手術閾值說明
Overjet (OJ)≤ -4.73 mm嚴重反咬
Wits appraisal≤ -11.18 mm嚴重骨骼 Class III
L1-MP angle≤ 80.8°下門牙已過度後傾（代償到極限）
Mx/Mn ratio≤ 65.9%上下顎長度比嚴重失衡
手術的一般適應症：
1. 骨骼差異嚴重（ANB 過大或過小）無法單靠矯正代償
2. Vertical excess（gummy smile / open bite 嚴重）
3. 顏面不對稱（asymmetry）
4. Obstructive sleep apnea（OSA）→ mandibular advancement

LeFort I osteotomy：上顎整體切開，可做前移、後移、上移（impaction）、下移、傾斜調整

BSSO vs BVRO

BSSO：下顎 ramus 矢狀劈開，可前移/後移，rigid fixation，IAN 損傷風險較高
BVRO：下顎 ramus 垂直切開，僅能後移，需 IMF 固定 6-8 週，IAN 損傷風險較低
你
BSSO 和 BVRO 都是做下顎的，有什麼差別？
師
差很多！BSSO 是切 sagittal（矢狀），BVRO 是切 vertical（垂直）。兩者的適應症、固定方式、併發症都不一樣。
項目BSSOBVRO

移動方向前移 + 後退都可以主要後退（setback）
固定方式Rigid fixation（plate + screw）→ 不需 IMFWire fixation → 通常需要 IMF（6-8 週）
神經損傷IAN（下齒槽神經）損傷風險較高IAN 損傷風險較低

陷阱題
「BSSO 最主要的併發症是什麼？」→ IAN（下齒槽神經）損傷（暫時性 ~30-50%，永久性 ~5%）。
「哪個手術比較適合有 TMD 的病人？」→ BVRO，因為 condyle 可以自由定位，不被 rigid fixation 限制。

穩定度排名：
上顎上移（impaction）最穩定 → 上顎前移 → 下顎前移 → 下顎後退 → 上顎下移最不穩定

術前矯正與 Surgery First

師
傳統正顎手術的流程是：術前矯正（12-18 個月）→ 手術 → 術後矯正（6-12 個月）。術前矯正的目的是 decompensation（去代償）。
Decompensation（去代償）：牙齒為了代償骨骼不正，會自行傾斜來維持咬合。
例如：Class III 病人的下門牙會自然 retrocline、上門牙會 procline → 代償骨骼差異
術前矯正要把這些代償拿掉 → 讓牙齒回到「正確的骨骼角度」→ 手術後才能咬得正
結果：術前矯正期間咬合會暫時變更差（因為去掉代償了），病人要理解這是正常的。

Surgery First Approach

你
聽說有 Surgery First 不用做術前矯正？
師
對，但不是每個人都適合。Surgery First 是先手術再矯正，優點是治療時間短（手術後利用 RAP 加速牙齒移動），但有嚴格的適用條件。
Surgery First 適用條件
輕度擁擠（minimal crowding）
Curve of Spee 平坦
上下門牙角度接近正常（不需大量 decompensation）
可以用手術定位預測最終咬合
Stabilization wire：術前矯正最後階段會放上粗的 SSW（如 0.019x0.025" SSW）作為穩定弓線，手術中用來做 IMF（intermaxillary fixation）。

術後照護與 Distraction Osteogenesis

Distraction Osteogenesis：骨切開後漸進牽張（1 mm/天），Latency → Distraction → Consolidation 三階段
時間照護重點
術後 1-2 天IMF（顎間固定）或 guiding elastics；流質飲食
術後 1-2 週改軟質飲食；開始輕柔張口練習
術後 2-4 週漸進增加食物硬度；持續 elastics（引導咬合）
術後 4-6 週移除 IMF（如果有用）；正常飲食
術後 6-8 週開始術後矯正
術後 3-6 個月骨癒合穩定；追蹤 X 光

Distraction Osteogenesis（DO，牽引成骨）

階段名稱時間內容
1Latency（潛伏期）術後 5-7 天等待初始骨痂形成，不動
2Activation（活化期）每天 0.5-1.0 mm緩慢牽開 → 新骨在間隙中生成
3Consolidation（固化期）8-12 週新生骨鈣化成熟；裝置不動
DO 的優點：可以移動骨骼超過傳統手術的限制量，軟組織隨之漸進延展，復發率低。
常用於：hemifacial microsomia、Pierre Robin sequence、嚴重下顎後縮。

正顎手術考試重點

考試重點框
  • Class III 手術閾值：OJ ≤ -4.73、Wits ≤ -11.18、L1-MP ≤ 80.8°、Mx/Mn ≤ 65.9%
  • LeFort I = 上顎（可全方向移動）；BSSO = 下顎矢狀劈開（最常用）；BVRO = 垂直切
  • BSSO 可前移+後退，用 rigid fixation；BVRO 主要後退，用 wire fixation + IMF
  • BSSO 主要併發症 = IAN 損傷；有 TMD 者優先考慮 BVRO
  • 穩定度：上顎 impaction 最穩定 → 上顎 downward 最不穩定
  • 下顎後退要注意 OSA 風險
  • 術前矯正 = decompensation（去代償），咬合會暫時變差
  • Surgery First 條件：輕度擁擠 + 平 Spee + 門牙角度正常
  • DO 三階段：Latency（5-7天）→ Activation（0.5-1mm/天）→ Consolidation（8-12週）
