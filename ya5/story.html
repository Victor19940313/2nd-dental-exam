CHAPTER 1

缺牙的開始

為什麼要做假牙？三條路怎麼選？
明
陳醫師，我左下後面少了兩顆牙，右邊也少了一顆，吃東西都只能用前面咬。到底要怎麼辦？
師
阿明你先別緊張。缺牙不處理的話，旁邊的牙齒會慢慢倒下來、對面的牙齒會往下長（過度萌發），咬合會整個亂掉。我們有三條路可以走。

三大假牙選項

想像你家客廳少了幾片地磚：

生活化比喻
活動假牙（RPD）= 鋪一塊地毯蓋住缺口，每天可以拿起來清洗
固定假牙（FPD）= 重新黏上新地磚，跟旁邊的磚緊緊相連
全口義齒（CD）= 整個地板重鋪，因為一片磚都不剩了
選項英文特點適合誰
活動局部義齒RPD
(Removable Partial Denture)		可自行摘戴，有金屬骨架缺多顆但還有自然牙的人
固定義齒FPD
(Fixed Partial Denture)		黏在支柱牙上，不能自行取下缺 1-2 顆，旁邊牙齒狀況好
全口義齒CD
(Complete Denture)		整副假牙，靠吸附力固位全口無牙的人
你
那植牙呢？
師
植牙（Implant）是第四條路，把鈦金屬螺絲打進骨頭裡當人工牙根，上面接假牙。國考會考一些基本概念，但重點還是在 RPD、FPD、CD 這三大類。我們一個一個來。
📋 考試重點框
  • RPD = 可摘式，有金屬骨架；FPD = 固定式，黏在支柱牙上；CD = 全口無牙用
  • 缺牙不處理的後果：鄰牙傾斜、對咬牙過度萌發、咬合紊亂
  • 全口義齒的咬合目標是「平衡咬合」，天然牙的目標是「MPO」（後面章節會詳述）
CHAPTER 2

嘴巴的地圖

Kennedy 分類法 — 幫每張嘴巴取名字
師
阿明，在幫你設計活動假牙之前，我得先幫你的嘴巴「分類」。就像醫院急診有五級檢傷，缺牙也有分類系統，叫做 Kennedy Classification。
你
為什麼要分類？
師
因為不同的缺牙分布，假牙的設計邏輯完全不同。分類就像是地址，告訴技師：這個人的嘴巴長什麼樣。

Kennedy 四大類

記住一個口訣：「雙游、單游、中間夾、前牙缺」

Kennedy 四大分類示意圖 — 粉紅虛線框 = 缺牙區，米色實框 = 現存牙
分類缺牙型態口訣
Class I雙側後方都缺牙（雙側游離端）I = 一對游離端
Class II單側後方缺牙（單側游離端）II = 一邊游離
Class III缺牙前後都有牙齒（支持端）III = 中間夾
Class IV前牙橫跨中線缺牙IV = 前牙缺
比喻時間
想像一排書架上的書：
Class I = 書架兩端的書都掉了，只剩中間 → 兩邊都「懸空」
Class II = 只有右邊的書掉了 → 一邊懸空
Class III = 中間抽掉幾本，但兩端還有書夾住 → 「有支撐」
Class IV = 書架正中央那排書全不見了 → 跨越中線

Modification（修飾區）

你
如果一個人又缺後面、又缺前面怎麼辦？
師
好問題！主分類看的是最遠端（最後方）的缺牙區。其他額外的缺牙區叫做「Modification」，只數區數，不數缺幾顆。
例如：右後方缺兩顆（游離端）+ 左邊中間缺一區 + 另一處又缺一區
→ 最遠端 = 右後方游離端 → 主分類 = Class II
→ 額外兩個缺牙區 → Modification 2
→ 結果：Kennedy Class II, Mod. 2

Applegate 八大法則（精華版）

法則白話翻譯
法則 4最後方的缺牙區決定主分類（最常考！）
法則 6Class IV 絕對沒有 Modification
法則 7Modification 只數「區數」，不數「顆數」
法則 8不存在 Class V（看到就是錯的選項）
法則 1分類要在拔牙之後才決定
法則 2第三大臼齒缺了且不補，可以不算
法則 3第三大臼齒如果當支柱牙，就要算進去
法則 5主分類以外的缺牙區 = Modification
⚠ 超級陷阱
前牙全缺 + 後牙單側缺 → 你可能直覺選 Class IV，但錯！
要看最遠端的缺牙在哪裡 → 如果最遠端是後牙 → 答案是 Class I 或 Class II，不是 Class IV！
📋 考試重點框
  • Kennedy 分類看最遠端缺牙區決定主分類（Applegate 法則 4）
  • Class IV 沒有 Modification（法則 6，鐵則）
  • Modification 數「區」不數「顆」（法則 7）
  • 不存在 Class V（法則 8）
  • 高頻考古題：108-2 #28 / 110-2 #28 / 111-2 #34 / 113-2 #29
CHAPTER 3

活動假牙的骨架

RPD 的三大要素、卡環、RPI 系統
師
阿明，你的嘴巴分類結果是 Kennedy Class II, Mod. 1。右下後牙缺兩顆（游離端），左上中間也缺一顆。我們來設計活動假牙的「骨架」。
你
骨架？活動假牙不就是一塊粉紅色的東西嗎？
師
那只是表面！裡面其實有精密的金屬骨架。設計骨架的核心就是三個字：支撐、固位、穩定。

RPD 三要素

比喻：想像一把椅子
支撐（Support）= 椅腳 → 防止你往下坐穿（義齒防沉陷）
固位（Retention）= 椅子的扶手 → 防止你被往上抬走（義齒防脫落）
穩定（Stability）= 椅背 → 防止你往旁邊倒（義齒防位移）
RPD 三要素力量方向示意 — 支撐防下沉、固位防脫落、穩定防側移
要素抵抗方向靠誰提供
支撐 Support抵抗下沉力（↓）Rest（支撐座）— 小匙子狀的金屬釘放在牙齒咬合面的凹窩上
固位 Retention抵抗脫落力（↑）卡環（Clasp）的末端 1/3 伸入牙齒倒凹區
穩定 Stability抵抗側向力（↔）主連接體 + 卡環的剛性臂（近心臂，在倒凹上方）
⚠ 國考超級陷阱
卡環的近心臂在倒凹上方 → 提供穩定，不提供固位！
只有遠心臂的末端 1/3 進入倒凹 → 才提供固位。
題目問「卡環哪個部分提供固位？」答案永遠是末端 1/3，不是整條卡環！

Survey Line（接近線）

你
什麼是倒凹？卡環怎麼知道要伸到哪裡？
師
我們用一個叫 Surveyor 的工具，把模型固定好之後，用一根針去描牙齒最寬的一圈線，這條線叫 Survey Line（接近線）。線的上面叫「倒凹上方」，線的下面就是「倒凹」。
Survey Line 將牙齒分成兩區 — 剛性臂在上方（穩定），固位末端在下方（固位）
接近線以上：放卡環的剛性臂（提供穩定）
接近線以下：放卡環的固位臂末端（提供固位）

RPI 系統 — 游離端的最佳設計

師
阿明右邊是游離端缺牙（Class II），後面沒有牙齒可以靠。這時候假牙咬東西會往黏膜方向沉下去，如果設計不好，支柱牙會被往後扯壞。所以我要用一套特別的系統：RPI。
R
Rest
放在支柱牙的近心窩
P
Proximal Plate
鄰接板要短！可以滑動
I
I-bar Clasp
從下方進入頰側倒凹

為什麼 Rest 要放近心？

假牙受力會以支點線為軸旋轉下沉。如果 Rest 放在遠心，旋轉的支點就直接壓在支柱牙上面，產生遠心扭力 → 支柱牙被「扳」壞！

放在近心，力量的傳導路徑改善，應力分散到更多結構，支柱牙安全很多。

I-bar 的禁忌症

禁忌原因
深軟組織倒凹I-bar 從下方進入，軟組織倒凹會卡住
前牙高美觀需求I-bar 在頰側可見
開口度受限摘戴角度不足

間接固位體（Indirect Retainer）

你
假牙後面沉下去，前面不就翹起來了嗎？
師
正是！所以我們需要一個東西壓住前面，防止翹起。這就是間接固位體。想像翹翹板，一端下沉，你需要在另一端坐一個人壓住。而且那個人要坐得越遠越好，力臂越長，壓的力量越大。
⚠ 國考超高頻陷阱
「間接固位體距支點線越近越好」→ 完全錯誤！
槓桿原理：力臂越長 → 抵抗翹起的力越大 → 間接固位體應盡量遠離支點線！

主連接體（Major Connector）

主連接體就像假牙的「背脊」，把所有零件連在一起。

下顎特點
Lingual Bar（舌側桿）首選！覆蓋黏膜最少。離齦緣至少 6mm
Lingual Plate（舌板）口底太淺用不了舌側桿時的替代
上顎特點
Single Palatal Strap缺牙不多時，覆蓋少、舒適
Complete Palatal Coverage牙槽骨嚴重吸收時，最大支撐
Horseshoe（馬蹄形）剛性最差！考題最愛問
📋 考試重點框
  • 三要素：Support（↓防沉）= Rest；Retention（↑防脫）= 卡環末端 1/3；Stability（↔防位移）= 剛性臂
  • RPI：游離端首選。R = Mesial Rest、P = 短板、I = I-bar。Rest 在近心防支柱牙扭力
  • I-bar 禁忌：深軟組織倒凹、高美觀需求、開口受限
  • 間接固位體越遠離支點線越好（槓桿原理）
  • Horseshoe 是上顎主連接體中剛性最差的
  • 下顎主連接體離齦緣至少 6mm
CHAPTER 4

沒有牙的世界

全口義齒 — 從印模到排牙
師
今天來了另一位患者，王奶奶，80 歲，全口無牙。RPD 需要有牙齒可以勾，但王奶奶一顆都沒有了。我們要做的是全口義齒（Complete Denture）。
你
沒有牙齒可以卡，全口義齒靠什麼固定？
師
靠三股力量：吸附力（唾液薄膜的表面張力）、邊緣封閉（義齒邊緣的密封效果）、和周圍肌肉的平衡。所以每一步都要非常精確。

製作流程

初步印模 → 用 Alginate 快速取概形 → 做出個別托盤（Custom tray）
邊界鑲嵌（Border Molding） → 用 Compound 一段一段調整托盤邊緣，讓患者做各種動作
最終印模 → 用 ZOE paste 或矽膠精確取模 → 翻出工作模型
決定咬合關係 → VD（垂直高度）+ CR（正中關係）
排牙 → 在咬合器上排人工牙 → 試戴蠟型
裝入 → 加工完成 → 裝戴、調整

咬合垂直高度（VD）

你
什麼是 VD？為什麼這麼重要？
師
VD 就是上下顎之間的「高度」。想像你的臉是一台手風琴，VD 就是手風琴打開的程度。太高，嘴巴關不攏、肌肉痠痛；太低，嘴巴塌陷、說話含糊。
RVD（放鬆）vs OVD（咬合）— 兩者的差就是 Freeway Space
RVD
Rest Vertical Dimension
肌肉完全放鬆時的高度
OVD
Occlusal VD
牙齒咬合時的高度
FWS
Freeway Space
= RVD - OVD
正常 2-4 mm
Freeway Space（自由間隙）= 2-4 mm，這是考試必背數值。
OVD 過高（FWS < 2mm）→ 牙齒撞擊、肌肉痠痛、TMD
OVD 過低（FWS > 4mm）→ 嘴唇塌陷、說話不清

正中關係（CR）

CR 的現代定義：下顎髁突在關節窩最上前方的位置（superior-anterior），是可以重複找到的起始位。

天然牙：CR 比 CO 略後 1-2mm（不完全重合）
全口義齒的目標：CR = CO（讓正中關係和正中咬合完全一致）

排牙原則與平衡咬合

你
天然牙不是追求 MPO（犬齒保護咬合）嗎？全口義齒為什麼不同？
師
因為全口義齒底下沒有牙根抓著骨頭，它就放在黏膜上面。如果側方運動時只有一邊碰到（像 MPO 那樣），另一邊就翹起來了！所以全口義齒要平衡咬合 — 在所有咬合動作中，兩邊同時有接觸，義齒才穩。

義齒性口腔炎（Denture Stomatitis）

長期戴全口義齒不好好清潔，上顎黏膜會出事。罪魁禍首：念珠菌（Candida albicans）。

Newton 分類表現
Type I針孔狀局部紅點（pinpoint）
Type II整片均勻發紅（diffuse）
Type III乳頭狀增生（papillary hyperplasia）→ 需手術切除
最重要的危險因子：夜間不摘除義齒（不是基托過度延伸！）
治療：清潔衛教 + 夜間摘除 + Nystatin 抗真菌 + 必要時重做義齒

Post Dam 與 Retromolar Pad

結構位置功能
Post Dam硬顎與軟顎交界後方 1-2 mm上顎義齒後緣的邊緣封閉（吸住的關鍵）
Retromolar Pad下顎最後磨牙後方的脂肪墊下顎義齒後緣的參考標誌（覆蓋前 1/2）
📋 考試重點框
  • Freeway Space = 2-4 mm（RVD - OVD）
  • OVD 過高 → 肌肉痠痛、TMD；過低 → 嘴唇塌陷
  • 全口義齒目標：CR = CO + 平衡咬合（不是 MPO）
  • 邊界鑲嵌用 Compound，最終印模用 ZOE paste
  • Denture Stomatitis：念珠菌感染，Newton 三型，最大危險因子 = 夜間不摘除
  • Post Dam 位於硬軟顎交界後 1-2 mm

固定的藝術

牙冠預備、Pontic、黏合劑
師
阿明左邊缺的那顆牙，前後都有健康的牙齒。這種情況最適合做固定牙橋（FPD）— 磨兩邊的牙齒做牙冠，中間接一顆假牙橋體。不用摘戴，跟真牙一樣。
你
磨牙齒要磨多少？有什麼原則？

削太直（taper 太小）→ 帽子套不上去

橋體（Pontic）設計

你
中間那顆假牙（橋體）底部跟牙肉的關係是什麼？
師
這是設計的關鍵！底部接觸方式不同，清潔和美觀差很多。
四種 Pontic 橫截面比較 — 注意底部與牙嵴的接觸方式差異
類型接觸方式特色
Sanitary完全不接觸黏膜最好清潔，但不美觀 → 下顎後牙首選
Modified Ridge Lap只接觸頰側，舌側開放美觀+清潔折衷 → 前牙區最常用
Ridge Lap頰舌兩側都包覆清潔死角 → 幾乎不再使用
Ovate伸入拔牙位凹陷最自然的齦緣，但需手術預備

Pier Abutment 問題

如果做一座五顆牙的長橋（例如 #2-#4-#6），中間 #4 就是 Pier Abutment。問題是：兩端咬合力不平衡，以 #4 為支點，一端翹起→黏合劑脫落。

解決方案：Non-rigid connector（非剛性連接體）
在 Pier 的遠心側（distal）設一個可以微動的 key-keyway 接頭。
放遠心 = 正確 / 放近心 = 錯誤。

黏合劑（Cement）

種類特點注意
Zinc Phosphate傳統首選，固位佳酸性強，刺激牙髓
GIC釋放氟，化學鍵結強度較低
Resin Cement固位力最強全瓷冠必用
ZOE臨時黏著丁香油抑制 Resin 聚合！
⚠ 重要陷阱
臨時冠如果用了 ZOE cement → 換永久 Resin cement 之前，必須徹底清除 ZOE 殘留，否則 Resin 聚合不完全！
Frozen Slab Technique：在冰冷玻璃板上調 Zinc Phosphate cement → 延長 Working time，但 Setting time 不變。
📋 考試重點框
  • Taper 理想 6°，軸壁高度 ≥ 3.5-4mm
  • Feather edge 最易 overcontouring，Chamfer 最常用
  • Die spacer 20-40 μm，離 margin 1mm 上方才塗
  • 前牙 Pontic 首選 Modified Ridge Lap，後牙首選 Sanitary
  • Pier Abutment 的 Non-rigid connector 放在 Pier 遠心側
  • ZOE 的丁香油抑制 Resin 聚合
  • Resin Cement 固位力最強 → 全瓷冠必用
CHAPTER 6

材料的秘密

印模材、陶瓷、包埋鑄造
你
前面一直提到各種印模材料，到底差在哪裡？
師
印模材就像相機 — 有手機相機（Alginate，方便但畫質普通），也有專業相機（Addition Silicone，精確但貴）。重點是：每種材料都有自己的「個性」和「地雷」。

四大彈性印模材

材料精確度穩定性副產物地雷
Alginate低差水取完要立即灌模
Polyether最高好無親水性 → 吸水膨脹，別泡水！硬 → 倒凹多時難取出
Addition Silicone最高最佳氫氣乳膠手套的硫抑制聚合！灌模前等 >30 分鐘排氫氣
Condensation Silicone中差酒精酒精揮發→收縮，1hr 內灌模
⚠ Addition Silicone 的兩大地雷
1. 乳膠手套含硫 → 接觸到矽膠表面會軟化不硬化 → 必須改用 vinyl 或 nitrile 手套
2. 副產物氫氣 → 剛取完印模內有氣泡 → 灌模前等 30 分鐘以上
精確度排名記憶法
Addition Silicone = Polyether > Condensation > Alginate
（加聚 = 聚醚 > 縮聚 > 藻）

陶瓷材料

你
PFM、全瓷冠、Zirconia，到底怎麼分？
師
想像建築：PFM 是鋼骨結構外面貼瓷磚，全瓷是整棟都用玻璃蓋，Zirconia 是用超級硬的人造鑽石做結構。
材料強度美觀適用
Feldspathic（長石質）低最佳前牙貼面
Leucite（IPS Empress）中佳前牙單冠
Lithium Disilicate（e.max）中高佳前後牙單冠、短橋
Zirconia最高傳統較差後牙冠橋、全口重建
Zirconia 為什麼這麼強？ — Stress-Induced Phase Transformation（應力誘發相轉換）
裂縫尖端有應力 → 四方晶轉為單斜晶 → 體積膨脹 → 壓縮裂縫 → 裂縫停止擴展 → 自我強化！

PFM 關鍵數值

燒附前焊溫度1110-1127°C
燒附後焊溫度710-743°C
燒結體積收縮> 20%
Metal coping 表面要光滑（增加 wetting，利於瓷結合）

包埋材料

種類適用
Gypsum-bonded（石膏結合型）金合金（低熔點，< 700°C）
Phosphate-bonded（磷酸鹽結合型）Ni-Cr、Co-Cr、PFM framework（高熔點，> 700°C）
📋 考試重點框
  • Addition Silicone：乳膠手套含硫 = 禁忌，灌模前等 30 分鐘排氫氣
  • Polyether：親水性最佳但不能泡水（吸水膨脹）
  • Zirconia 強度高的原因：應力誘發相轉換（tetragonal → monoclinic → 體積膨脹壓裂縫）
  • PFM 燒結收縮 > 20%，coping 要光滑
  • Gypsum-bonded → 金合金；Phosphate-bonded → PFM / 高熔點合金
CHAPTER 7

咬合的平衡

MPO、咬合器、Hanau's Quint
你
前面提到天然牙要 MPO、全口義齒要平衡咬合，到底差在哪裡？
師
這是整個咬合學的核心，讓我好好解釋。

MPO（Mutually Protected Occlusion）

MPO 又叫犬齒保護咬合，是天然牙的黃金標準。核心邏輯是「互相保護」：

比喻：日夜班保全
正中咬合（CO）時：後牙接觸、前牙不碰 → 後牙當「日班保全」，保護前牙不被垂直力磨耗
側方/前突運動時：前牙（犬齒）引導、後牙全分離 → 犬齒當「夜班保全」，保護後牙不受側向力破壞
MPO = 互相保護咬合：CO 時後牙咬、側方時只有犬齒引導

MPO 四大條件

#條件
1CR = CO（正中關係 = 正中咬合）
2CO 時：後牙接觸、前牙不接觸
3側方及前突運動時：只有犬齒接觸，後牙全部分離
4牙周健康、Angle Class I
⚠ 常見陷阱
「側方運動時，工作側後牙有接觸」→ 這是 Group Function（群功能咬合），不是 MPO！
MPO = 只有犬齒引導 → 工作側和非工作側後牙都不接觸。

MPO 不適用的三種情況

Angle Class II / III前牙關係異常，犬齒無法引導
全口義齒改用平衡咬合
嚴重磨牙症犬齒磨耗，引導功能喪失 → 改用 Group Function

Hanau's Quint（漢諾五因素）

這是控制全口義齒平衡咬合的五個關鍵旋鈕：

Condylar Guidance
髁突引導角
不能改
2

不能改
3

4

5

五因素中，1 和 2 由患者的解剖結構決定，不能改。
3、4、5 可以透過選牙和排牙來調整。
記法：前兩個（跟人體有關的）不能改，後三個（跟假牙有關的）可以調。

咬合器（Articulator）

Class名稱功能
ISimple只能開閉口（像鉸鏈）
IISemi-adjustable可模擬部分下顎運動 → 臨床最常用

Arcon = 髁球在下顎（anatomically correct）
Non-Arcon = 髁球在上顎（相反）
Arcon 較精確，代表：Whipmix、Denar
Non-Arcon 代表：Hanau
📋 考試重點框
  • MPO 四條件：CR=CO、CO 時後牙碰前牙不碰、側方只有犬齒引導、Class I + 牙周健康
  • 側方運動工作側後牙有接觸 = Group Function（不是 MPO）
  • Hanau's Quint：前兩個（Condylar + Incisal guidance）不能改，後三個可調
  • 全口義齒 = 平衡咬合（兩邊同時接觸）；天然牙 = MPO
  • Arcon（髁球在下顎）= 符合解剖 = 較精確
CHAPTER 8

關節的煩惱

TMD 分類、診斷與治療
明
陳醫師，我最近張嘴有喀喀聲，有時候嘴巴張不開，是怎麼了？
師
阿明，這可能是顳顎關節紊亂症（TMD）。讓我檢查一下。

TMD 四大類型

類型特徵關鍵鑑別
肌肉性（Myofascial pain）咀嚼肌痠痛、壓痛沒有關節聲音
關節盤前移位 WITH reduction張口時 Click（彈響）Click 後功能正常（盤子有回位）
關節盤前移位 WITHOUT reduction嘴巴張不開（< 40mm）「閉鎖」（Closed lock），沒有 click
退化性（Osteoarthritis）Crepitus（骨擦音）X 光可見骨變化
Click（彈響）= 關節盤快速移位的聲音 → With reduction
Crepitus（摩擦音）= 骨對骨摩擦 → Osteoarthritis 或盤穿孔
正常張口度 ≥ 40mm，< 40mm = limited opening

治療原則：由保守到積極

衛教 + 自我護理（熱敷、軟食、不要大張口）
咬合板治療（Night guard）
物理治療（肌肉放鬆、超音波）
藥物（NSAIDs、肌肉鬆弛劑）
關節腔注射（透明質酸、類固醇）
手術（關節鏡 → 開放手術，最後手段）
📋 考試重點框
  • With reduction = Click；Without reduction = Closed lock（張不開、沒有 Click）
  • Crepitus = 骨擦音 = Osteoarthritis
  • 正常張口 ≥ 40mm
  • 治療順序：保守 → 積極（衛教 → 咬合板 → 物理 → 藥物 → 注射 → 手術）
  • TMD 是多因素疾病，單一咬合因素不一定直接導致

Biological Width

比喻：牙齒的「安全距離」
想像牙齒是一棟房子，牙齦是圍牆。圍牆跟房子之間有一段「不能動」的安全距離 — 這就是 Biological Width。如果你的修復體邊緣侵入了這個距離，圍牆就會開始崩壞（牙周發炎、骨吸收）。

0.5~3mm JE 0.97mm CT 1.07mm Biological Width ← 牙槽嵴頂 → 至少 ~3mm Sulcus + BW 修復體邊緣 不可侵入 BW！
Biological Width 結構示意 — JE + CT = 2mm，加上齦溝至少 3mm 的安全距離

Munsell 色彩系統三要素

參數定義牙科意義

Value（明度）亮暗程度最影響美觀，應優先確認
